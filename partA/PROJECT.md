# Part A — Project Definition

## Project Title
Personal Task Tracker (AI-Assisted)

## Topic Choice
Option 2: Personal task tracker — task CRUD, due date, priority, label, search/filter.

## Goal
Build a small task management application to practice AI-assisted software construction workflow:
Spec → Generate → Review → Integrate.

## Scope (Planned Features)
### Core Features (Must Have)
1. Create a task
2. List tasks
3. Update a task
4. Delete a task
5. Search tasks by keyword
6. Filter tasks by status and priority

### Additional Features (Nice to Have)
- Label/tag support
- Sorting by due date

## Out of Scope
- Authentication / login
- Multi-user accounts
- Cloud deployment

## Non-functional Requirements
- Input validation for all API requests
- Unit testing (>=10 tests)
- OpenAPI 3.0 spec
- Clean separation (routes/controllers/services)

## Deliverables for Part A
- Architecture diagram (Mermaid)
- Stack comparison
- ADR-001
- CLAUDE.md rules
- AI planning session log
