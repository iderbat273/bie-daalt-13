# Part A — Stack Comparison

## Option 1: Node.js + Express + TypeScript
### Pros
- Fast development
- Huge ecosystem
- Easy REST API
- TypeScript improves safety
- Works well with Jest testing

### Cons
- Express is flexible and can become messy without discipline

---

## Option 2: Python + FastAPI
### Pros
- OpenAPI auto-generated
- Strong validation (Pydantic)
- Clean API design

### Cons
- Different tooling compared to Node ecosystem

---

## Option 3: Java + Spring Boot
### Pros
- Strong enterprise architecture
- Dependency injection and structure

### Cons
- Overkill for small project
- More boilerplate

---

# Decision
## Selected Stack
Node.js + Express + TypeScript

## Reason
This stack is chosen because it is simple, fast to implement, easy to test,
and suitable for demonstrating AI-assisted workflow with good Git history.
