# CLAUDE.md — Project Rules

## Build Commands
npm install
npm run dev

## Test Commands
npm test

## Conventions
- TypeScript
- REST API
- routes/controllers/services/repository separation
- No business logic in controllers

## Git Conventions
- Conventional Commits (feat, fix, docs, test, refactor, chore)
- Small commits frequently
- Spread commits across multiple days

## No-Go Zones
- Do not hardcode secrets
- Do not skip validation
- Do not commit huge code dumps
- Do not add authentication unless required
