# Self Evaluation

## 1. Could I reproduce this project without AI?

- Partially.

I could build basic Express CRUD API manually, but:
- Jest + TypeScript configuration would take longer
- OpenAPI setup would require documentation reference
- Debugging module issues would be slower

So my answer is: **Partially**

---

## 2. What would I do differently?

- Start with simpler TypeScript config (avoid ESM complexity)
- Add testing setup earlier in the project
- Commit more frequently with smaller changes
- Define architecture more clearly before implementation

---

## 3. What did I learn?

- How AI can accelerate backend scaffolding
- Importance of verifying AI-generated code
- Debugging Node.js + TypeScript + Jest integration issues
- Git workflow discipline (branching + commits)
- API testing using curl/Postman

Overall, I learned how to work with AI as a development assistant rather than relying on it blindly.
